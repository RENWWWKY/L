> **适用工具**：Midjourney、Stable Diffusion、DALL·E、即梦、可灵等文生图；图生图时可垫 `程予安个人剧本封面.png` 作风格参考。

## 一、统一视觉规范（全角色共用）

### 1.1 画风与氛围

| 维度 | 说明 |
|------|------|
| **画风** | 半写实韩漫 / 国漫精品插画；线条利落，肤质与布料有细节；**非**真人照片、**非** 3D 渲染 |
| **色调** | 午夜蓝、深灰、黑为主；霓虹红/蓝/紫作点缀；人物肤色偏冷白 |
| **光线** | 低调戏剧光：窗外城市霓虹 + 室内单侧暖灯（如台灯）形成明暗对比 |
| **场景** | 玻璃湾七号高层落地窗内；窗外暴雨、雨痕糊玻璃、远处摩天楼灯火 bokeh |
| **构图** | 半身至腰际中近景；人物略偏画面中心；上方留标题区，下方留姓名区 |
| **天气** | 玻璃上可见雨流、水珠；人物发梢/肩线可带微湿感（除周启外均可） |

### 1.2 封面排版元素（后期可 PS，也可写进提示词）

- **顶栏标题**：「雨夜归零」——白色泼墨/毛笔大字，笔画边缘可带红色飞溅（非血腥写实）
- **副标题**：「剧本杀 · 角色剧本」——小字白或浅灰
- **底栏姓名**：角色全名——大号白色衬线体
- **底栏身份**：「男 \| ……」或「女 \| ……」——小号无衬线体
- **背景彩蛋**（可选）：远处霓虹牌「ZERO」、竖排「雨夜归零」蓝紫灯牌

### 1.3 统一英文风格前缀（可复制到每条提示词开头）

```
semi-realistic manhwa illustration, Korean webtoon premium character poster, cinematic low-key lighting, rainy night metropolis through floor-to-ceiling window, rain streaks on glass, neon city bokeh red and blue, dark midnight blue color palette, sharp detailed linework, dramatic shadows, vertical 9:16 portrait composition, NOT photorealistic, NOT 3D, NOT live action
```

### 1.4 统一负向提示词

```
photorealistic, real photo, 3D render, CGI, live action, horizontal landscape, cropped head, deformed hands, extra fingers, blurry face, cartoon chibi, bright daylight, sunny, cheerful, low quality, watermark, text gibberish
```

### 1.5 全角色共通【禁止】

- 禁止 Q 版、萌系、过度性感暴露
- 禁止明显西方面孔（需东亚面孔）
- 禁止现代休闲卫衣、T 恤、运动鞋（晚宴场景）
- 禁止手持武器、明显血腥伤口
- 林晚星、苏晚晴、沈知意：**禁止**男性化面部或短发男性造型
- 陆景川、程予安、周启：**禁止**女性化妆容或裙装

### 2.1 林晚星（东道主 · 非玩家）

| 项 | 内容 |
|----|------|
| **性别 / 年龄感** | 女 · 29–32 岁 |
| **对外身份** | 归零科技创始人 / 玻璃湾七号女主人 |
| **气场** | 控场、先饮后敬；笑浅而稳；女王式成熟 |

#### 五官与发型

- **脸型**：鹅蛋脸偏长，颧骨线条柔和但轮廓清晰；下颌收得干净，有「能镇住场」的成熟感
- **眉**：眉峰微挑，眉尾细长，自然深棕近黑，不浓眉
- **眼**：深棕近黑，眼型偏长，外眼角微微上挑；睫毛浓密；眼神温而利——对外人像谈判桌，对熟人有一丝疲惫
- **鼻**：鼻梁直挺，鼻尖精致，侧光下高光明显
- **唇**：唇形饱满，常态淡豆沙或酒红；唇线清晰；可带极淡唇印高光（呼应「先饮一口」）
- **肤**：冷白皮，妆面精致哑光；颧骨处略扫修容
- **发型**：深黑长发，低盘髻或半挽髻，两缕碎发垂颊侧；发髻可插简约金属发簪或珍珠发夹；发梢微湿

#### 服装与外在

- **主服装**：**墨色丝绒长晚礼服**——一字领或 modest V 领，长袖或七分袖，裙摆不在画面内（半身构图）；丝绒有微弱反光，像吸了雨夜的潮气
- **首饰**：小巧钻石或珍珠耳钉；腕上极细表链或手链；无夸张项链
- **鞋**（全身版可选）：黑色细高跟，不在半身构图中强制
- **道具**：一只高脚香槟杯，杯中浅金液体与细泡沫；或空手微抬做敬酒姿态
- **姿态**：微侧身对窗，另一手轻搭杯脚；背脊笔直，下巴微抬

#### 表情

- 嘴角浅笑，不达眼底；眉心极淡川字；整体「庆功夜的主人」，底下藏着一夜未签字的疲惫

- **脸型**：略短的鹅蛋脸，下颌Soft但线条干净；比林晚星更「冷」一点

- **姿态**：站姿笔直，肩略内收；目光偏下或侧看窗外，不直视镜头

#### 表情

- 克制、疏离；唇线微抿；骄傲里带一点旧情未了的钝痛，但不哭不闹
