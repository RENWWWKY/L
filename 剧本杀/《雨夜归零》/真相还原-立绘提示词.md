# 《雨夜归零》角色立绘 · AI 提示词（纯白底 · 全身）

> **用途**：生成供 App / 竖屏分镜垫图用的 **REF 立绘**（`ai-refs/REF_{角色}_全身_正面.png`）。  
> **构图**：**头顶到脚底完整入画**，正面站立，双脚可见，**背景纯白色**（#FFFFFF，无渐变、无阴影地、无场景）。  
> **画风**：2D 动漫 · 视觉小说 CG · 赛璐璐平涂 — 与 `真相还原-AI视频分镜.md` 一致；**非**真人照片。  
> **关联**：雨夜封面海报见 [`角色形象-AI封面提示词.md`](./角色形象-AI封面提示词.md)。

---

## 一、统一规范

### 1.1 构图与输出

| 项 | 要求 |
|----|------|
| **景别** | **全身**（full body）：头顶、发型、五官、躯干、四肢、**鞋尖**均完整，四边留约 5% 安全边 |
| **机位** | 正面平视（front view, eye level），人物居中，自然站立 |
| **背景** | **solid pure white background**, `#FFFFFF`, no gradient, no floor shadow, no scenery |
| **画幅** | 竖屏 **2:3** 或 **9:16**（推荐 832×1248 / 768×1152）；全身需留足上下空间 |
| **手部** | 自然垂放或轻握道具；五指清晰，勿插袋到不可见 |
| **命名** | `REF_林晚星_全身_正面.png` …（见下表） |

| 角色 | 输出文件名 |
|------|------------|
| 林晚星 | `REF_林晚星_全身_正面.png` |
| 苏晚晴 | `REF_苏晚晴_全身_正面.png` |
| 陆景川 | `REF_陆景川_全身_正面.png` |
| 沈知意 | `REF_沈知意_全身_正面.png` |
| 程予安 | `REF_程予安_全身_正面.png` |
| 周启 | `REF_周启_全身_正面.png` |

### 1.2 统一英文风格前缀（每条提示词开头）

```
2D anime illustration, visual novel character sprite, Japanese galgame standing CG, cel shading, flat color, clean lineart, full body front view, head to toe visible, both feet and shoes fully in frame, standing straight facing viewer, solid pure white background #FFFFFF, no background elements, no cast shadow on ground, even soft studio lighting on character only, NOT photorealistic, NOT 3D, NOT live action
```

### 1.3 统一负向提示词

```
photorealistic, real photo, 3D render, CGI, live action, cropped feet, cut off legs, missing shoes, head cut off, half body only, bust shot, waist up, sitting, walking, dynamic action pose, background scenery, gradient background, gray background, floor shadow, drop shadow, complex background, rainy night, city, window, neon, chibi, super deformed, low quality, blurry, watermark, text, logo
```

### 1.4 共通【禁止】

- 禁止背景出现任何场景、纹理、灰底、渐变
- 禁止裁切脚、裁切头顶
- 禁止 Q 版；禁止过度暴露
- 禁止西方面孔（东亚面孔）
- 禁止卫衣、T 恤、运动鞋（晚宴/制服设定）
- 封面专用元素（雨夜、霓虹、标题字）**不出现在立绘中**

### 1.5 半身立绘（可选 · 分镜部分镜头需用）

若分镜需 `REF_*_半身_正面.png`：在全身图基础上 **居中裁切至大腿中部**，或另生成时将前缀中 `full body` 改为 `half body waist up`，其余五官服装与下表一致，背景仍为纯白。

## 三、出图参数建议

| 工具 | 建议 |
|------|------|
| **分辨率** | 832×1248（2:3）或 768×1344（9:16）；务必检查脚底是否被裁 |
| **Midjourney** | `--ar 2:3 --style raw`；加 `full body character sheet` |
| **SD / ComfyUI** | 用「立绘 / standing / white background」LoRA；CFG 7；after gen 检查用魔棒抠图 |
| **后期** | 若 AI 带出灰底或地影，PS 魔棒 + 纯白 `#FFFFFF` 填充；必要时手动补全鞋尖 |
| **一致性** | 六人统一同一画风前缀；仅服装五官按上表区分 |
