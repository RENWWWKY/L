## 设计原则

- **角色称呼**：剧情叙述中（除第二人称「你」外），四人及林晚星均使用**全名**（林晚星、陆景川、苏晚晴、沈知意、程予安），不用「林总」「陆总」「晚晴」「景川」等简称；酒窖标签写作「陆景川贺酒」。若因语境必须用第三人称代词，林晚星、苏晚晴、沈知意用「她」，陆景川、程予安用「他」。
- **开场不公布真凶姓名**；玩家任务仅为「找出导致林晚星中毒的人并投票」。
- **真凶玩家开局也不知自己是凶手**；随幕推进在私密剧本中逐步恢复记忆、产生怀疑或确认。
- **各角色互不知晓他人秘密身份**（秘密恋情、匿名信、挪用等）；自我介绍与发言只基于你「当时以为」的表象。
- **公共剧情尾部 = 下一幕开头场景**；DM 宣读完过场后，各角色第一行须落在同一时空。
- **发言环节只给任务**，不写示范台词；由玩家与 AI 自由发挥。
- **每幕正文末附【本幕任务】**：对应该轮讨论（第一幕后 / 第二幕后 / 第三幕后），提示本轮发言**可往哪些方向聊**，尽量做到即可，不作硬性打卡；顶部【阶段任务】表为全程总览，不替代本幕任务。

### 林晚星（东道主 · 非玩家）

**气场**：玻璃湾七号的女主人，墨色长裙与腕表冷光；对外是「只谈好消息」的庆功姿态，对内是签字前夜的控场者。笑不深、不讨好，举杯有仪式感；中毒时从稳到碎，嘶哑、惊恐，但仍要保留「创始人」尊严。

**音色**：御姐音 / 成熟女王音（偏中低音区，不甜腻、不少女）。

```
【角色】林晚星，女，约30岁，科技公司创始人。
【音色】御姐音，成熟女王音，中低音区，声线有磁性，语速中等偏稳。
【气场】晚宴控场，笑浅而稳，礼貌但有距离；对合伙人温柔一句、对条款不卑不亢。
【说话方式】短句干脆，举杯、吩咐、安抚皆克制；危机时气息发紧、嘶哑，仍少哭喊。
【情绪】常态冷静自信 → 中毒时惊恐虚弱 → 医院回忆句含糊、像梦呓。
【场景】现代滨海私宅雨夜，轻奢，非古装。
【禁止】萝莉音、嗲音、夸张播音腔、喜剧感。
```

### 苏晚晴（玩家 · 女）

**气场**：「会讲故事的人」——品牌总监，前任归来，米色风衣、话不多。骄傲、克制，不主动讨好林晚星，也不演苦情；雨夜更像一枚被摆回棋盘的棋子，越被逼越清醒。

**音色**：少御音 / 清冷知性御姐（比林晚星略轻半档，更疏离）。

```
【角色】苏晚晴，女，约28岁，品牌总监，林晚星前任。
【音色】少御音，清冷知性御姐，成熟女声偏淡，中音区，气息稳。
【气场】疏离、骄傲、克制，不发嗲；委屈压在喉头，多用停顿代替哭腔。
【说话方式】句尾常收短；讽刺时轻、冷；辩护时逻辑清楚，不喊冤。
【情绪】表面平静 → 讨论时渐锐 → 听到医院电话时声涩仍控。
【场景】现代言情轻推理，雨夜客厅对峙。
【禁止】甜妹音、哭腔泛滥、撒娇、过于活泼。
```

### 陆景川（玩家 · 男 · 真凶）

**气场**：联合创始人 / CTO，话少、背直，合伙人式无害微笑。技术人的克制与禁欲感；心虚时唇线更紧、声线更低更硬，爱用短句推责、反问，不歇斯底里。

**音色**：青年低沉男声 / 禁欲系低音（男中低音，干净，少笑意）。

```
【角色】陆景川，男，约31岁，科技公司联合创始人兼CTO，真凶但前期不自知。
【音色】青年低沉男声，禁欲系低音，成熟男中音，偏低，气息沉。
【气场】话少、脊背直、社交微笑浅；压力下更冷更硬，不主动认罪。
【说话方式】短句、反问、推责时语速略快；否认时咬字清楚、声线发紧。
【情绪】第一幕「以为清白」的稳 → 第二幕动摇仍对外硬撑 → 第三幕苍白、唇抿死紧。
【场景】暴雨私宅，现代商战氛围，非警察审讯喊叫。
【禁止】油腻霸总、东北喜剧腔、少年奶音、过度凶吼。
```

### 沈知意（玩家 · 女）

**气场**：沈厚泽基金代表，丝绒裙、深色不抢眼；宣读条款时**声线温柔、字字冷**。父亲的「别让我失望」压在身上；恋陆景川但先藏，对外是资方刀刃，不是小女生。

**音色**：少御音 / 温柔冷感少女音（听感偏年轻于苏晚晴，但逻辑锋利）。

```
【角色】沈知意，女，约26岁，资方派驻代表，沈厚泽之女（对外不张扬）。
【音色】少御音，温柔冷感少女音，知性女声，中高音区清亮但克制。
【气场】条款即刀，礼貌外壳；冷笑时极轻，质问时一字一顿。
【说话方式】公开场合温柔、精确；私下电话压低气声；不谈感情时像尽调报告。
【情绪】克制专业 → 监控缺口时目光冷 → 第三幕父亲与律师压力下更白更硬。
【场景】现代滨海，签字与回购语境，非古风宫斗。
【禁止】软糯撒娇、纯甜少女、过于软糯的「妹妹音」。
```

### 程予安（玩家 · 男）

**气场**：林晚星四年助理，工牌与主卡是身份；餐边柜后的影子，低声应「是」。忠诚、机要、情绪深藏；被逼坦白时发白、喉结动，仍想护林晚星体面。

**音色**：青年干净男声 / 温和男中音（偏低半档，像「永远站在半步外」的人）。

```
【角色】程予安，男，约27岁，总裁助理，跟林晚星四年。
【音色】青年干净男声，温和男中音，偏低，气息稳，音量常不大。
【气场】沉默、利落、忠诚；服务感强，不抢话，关键句才亮。
【说话方式】日常极简；承认取信时声发白、句短；护主时不煽情。
【情绪】隐忍服务 → 被怀疑时紧 → 听到林晚星含糊话时指节发白、声颤但克制。
【场景】雨夜别墅，后勤与机要，非军人吼令。
【禁止】霸道总裁、油滑痞气、过于洪亮的播音腔。
```

### 周启（保安队长 · DM 代读）

**气场**：驻场安保负责人，手套与消毒水味；用词克制，不煽情，宣布留人、查监控、上楼查碎纸机时**权威而冷**。像刑侦现场指挥，不是反派咆哮。

**音色**：成熟男中低音 / 刑侦旁白感大叔音。

```
【角色】周启，男，约40岁，别墅保安队长，现场负责人。
【音色】成熟男中低音，刑侦旁白感，沉稳大叔音，语速稳，咬字清楚。
【气场】权威、克制、冷面公平；命令短，目光「刮过」人脸的压迫感用语气体现即可。
【说话方式】陈述句多，反问少；宣布规则时不抬高音量，靠停顿施压。
【情绪】全程专业冷静，偶有一句「希望还有人会说真话」略沉。
【禁止】军阀吼叫、黑社会腔、过度亲切唠嗑。
```

### DM 主持 / 旁白（欢迎词、公共剧情、结局）

**气场**：雨夜电影感，把玩家带进玻璃湾七号；不剧透真凶，节奏如纪录片旁白，到公共剧情时略收紧，结局时由紧到缓。

**音色**：叙述男中音或中性沉浸旁白（与周启区分：DM 更「讲故事」，周启更「执法记录」）。

```
【角色】剧本杀DM旁白，《雨夜归零》雨夜沉浸主持。
【音色】叙述男中音，或中性偏男沉浸旁白，中低音，气息深，语速中等。
【气场】电影开场白质感，雨声与海风在语气里；不轻浮、不念稿机器感。
【说话方式】公共剧情有画面感，对白引号内角色由各自声线演绎时，旁白只铺景；
  线索宣读时略干、像念证物清单；结局段可放慢半拍。
【情绪】欢迎温暖克制 → 公共剧情紧张 → 真相还原庄重。
【禁止】夸张相声、悬疑恐怖尖叫腔、过度活泼主持腔。
```

## 标准流程

1. 自主选角 → 仅阅读本角色剧本  
2. DM 欢迎词 + **背景（晚宴前）** + 终局任务（不点名真凶）  
3. 阅读「自我介绍」→ **公开发言任务**  
4. DM **公共剧情①**（至「客厅沙发、收走手机」）→ 各角色 **第一幕**（从沙发回溯今夜）  
5. **公共线索①** → 第一轮讨论任务  
6. DM **公共过场一**（22:00 还手机、查监控）→ **第二幕**（从过场尾部接续）  
7. **公共线索②** → 第二轮讨论任务  
8. DM **公共过场二**（翌晨医院来电、律师到门）→ **第三幕**（从过场尾部接续）  
9. **公共线索③** → 第三轮讨论任务 → 终局投票 → 结局 → 真相还原
