## 真相还原提示词规范（必读）

> **有对白**：垫图 + I2V 须写清 **动嘴型**（`张口说话`、`口型同步`）。  
> **禁止抽象画面**：不要双重曝光、符号叠化、意识流拼贴、看不懂的隐喻——**每一镜必须是具体地点 + 具体人物/道具 + 看得懂在发生什么**。

### 视角与信息面分离（硬性）

| 规则 | 说明 |
|------|------|
| **人脸 ≠ 屏幕/文件正面** | 角色 **正脸或半脸** 出镜时，电脑/手机/平板/文件夹 **只露背面、边框、或不可读侧角**；I2V 只做表情/呼吸/手势 |
| **内容特写单独切** | 邮件 UI、支付页、监控时间轴、合同条款、大屏图表等 **另起一镜**，**无人物、无正脸** |
| **鼓励多切镜头** | 同一情节用 **反应镜（人）→ 插入镜（物证/UI）→ 环境镜** 连切，比一镜塞满更稳、更像叙事短片 |
| **特写可用** | 人物 **面部特写、手部特写、物证特写** 均可；特写镜不写「人脸与可读屏幕/文件同框」 |

| 字段 | 说明 |
|------|------|
| **画面内容** | 中文：这一镜观众具体看到什么（给剪辑/审核用） |
| **入镜转场** | 从上一镜怎么进来 → **后期特效**（如硬切、淡入、闪回入） |
| **出镜转场** | 到下一镜怎么出去 → **后期特效** |

### 转场特效词表（后期用）

| 标注 | 后期做法 |
|------|----------|
| **硬切** | 直接切，无过渡 |
| **淡入 / 淡出** | 黑场 0.5s 淡入或淡出 |
| **叠化** | 两镜重叠渐变约 0.8s |
| **闪白** | 白场闪烁 0.2–0.4s |
| **闪回入** | 闪白或模糊放大 + **暖黄/复古/暗角/轻颗粒**（进入过去） |
| **闪回出** | 闪白或模糊收缩 + **去复古滤镜、回到冷蓝雨夜**（回到案发夜） |
| **交叉剪辑** | 同一时刻两地点硬切交替（注明 A/B 地点） |
| **快切** | 0.3–0.5s 一镜，连切 3–4 个 **具体特写**（每特写仍须看得懂） |

**T2I 负向**：`抽象，双重曝光，符号化，超现实拼贴，写实照片，3D，横屏16:9，Q版，文字不可读，人脸与可读屏幕同框，视角错乱，人物对镜头时文件正面可读`

**I2V 负向**：`说话时嘴闭合，口型不同步，面部变形，场景抽象化，屏幕内容融入人脸`
